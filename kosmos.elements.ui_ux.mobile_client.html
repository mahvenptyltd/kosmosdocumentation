km_press(locator, explicit_wait=None)¶

Description: | This mobile util method will press on the given value on the locator provided in mobile screen.

Parameters

  • locator (String) – This parameter will contain the locator in a fixed format which is , locatortype=locator . For eg. id=username or xpath=.//*[@id=’username’]

  • explicit_wait (Integer) – This parameter will by default be none and the number passed in it would be the explicit wait(in seconds) for the particular element

Returns

None

Examples:
self.obj_mobileclient_actions = MobileClientActions(team_config_filename, pdriver)
self.obj_mobileclient_actions.km_press(“xpath=//a[@class=’signIn’]”)
self.obj_mobileclient_actions.km_press(“xpath=//a[@class=’signIn’]”,30)

km_tap(locator, explicit_wait=None)¶

Description: | This mobile util method will tap on the given value on the locator provided in mobile screen.

Parameters

  • locator (String) – This parameter will contain the locator in a fixed format which is , locatortype=locator . For eg. id=username or xpath=.//*[@id=’username’]

  • TapType (String) – This parameter will contain the tap type : single tap or double tap on screen

  • explicit_wait (Integer) – This parameter will by default be none and the number passed in it would be the explicit wait(in seconds) for the particular element

Returns

None

Examples:
